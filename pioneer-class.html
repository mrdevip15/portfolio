📌 Ringkasan Proyek

Platform web tryout & pembelajaran untuk persiapan Akpol, Akmil, dan ujian akademi lainnya. Dibangun untuk memberikan simulasi ujian yang presisi dan manajemen data peserta yang komprehensif.

Satu Sesi / User

Keamanan data dengan mencegah login ganda di waktu bersamaan.

Tracking Aktif

Pemantauan status online peserta secara real-time.

Rate Limiting

Perlindungan endpoint API dari penyalahgunaan dan traffic berlebih.

Untuk Peserta

  • Dashboard: Pantau progress tryout, nilai, dan jadwal aktif.
  • Ujian Token: Input kode ujian unik untuk akses tryout privat.
  • Tes Kecermatan: Modul khusus tes kecepatan dan ketelitian.
  • Blog: Akses artikel edukasi akademi dengan konten dinamis.

Kemampuan Admin

  • Master Soal: Input soal, manajemen paket, dan cetak soal (PDF).
  • Token Genius: Generate & kelola masa berlaku kode ujian.
  • Rekap Nilai: Statistik hasil per peserta atau per token.
  • CMS Blog: Kelola konten artikel dengan editor terintegrasi.
