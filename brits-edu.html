Case Study

BritsEdu

Platform Tryout & Assessment end-to-end yang dirancang untuk mengelola ujian berskala besar dengan manajemen soal yang presisi dan sistem penilaian IRT.

Panel Siswa

  • Auth: Registrasi & Google OAuth 2.0.
  • Dashboard: Monitoring progress dan daftar tryout aktif.
  • Execution: Engine ujian dengan timer per subtest & review jawaban.
  • Outcome: Hasil instan, ranking nasional, dan unduh sertifikat terverifikasi.

Panel Admin

  • Management: Kontrol penuh atas periode, subtest, dan bank soal.
  • Bank Soal: Penugasan soal dengan filter kesulitan & kategori.
  • Settings: Kustomisasi IRT, sertifikat, dan rilis pembahasan.
  • Analytics: Laporan statistik mendalam & analisis tingkat kesulitan soal.

Technical Highlights

Sistem Skoring IRT

Implementasi skala 200–1000 dengan parameter gamma untuk transformasi non-linear. Perhitungan batch otomatis dengan progress tracking real-time.

Caching & Performa

Optimasi Redis untuk cache ranking nasional, bank soal, dan skor pengguna. Strategi invalidation cerdas untuk menjamin akurasi data.

Automasi Infra

Integrasi API Kamatera untuk scaling server otomatis (CPU/RAM) berdasarkan jadwal ujian guna optimasi biaya infrastruktur.

Backend Architecture

Pola Repository + Service yang bersih. Pemisahan middleware auth/error-handler dan dokumentasi API otomatis via Swagger.
